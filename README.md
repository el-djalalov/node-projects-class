# Node.js Beginner Projects — Class Repo

This is the shared repository for our Node.js projects. You'll build five projects here, one at a time, and submit each as a **pull request (PR)**.

## How this repo is organized

```
projects/        ← the assignment briefs (read-only — do not edit these)
  1-number-guessing/
  2-task-tracker/
  3-expense-tracker/
  4-github-activity/
  5-unit-converter/

submissions/     ← where YOU write your code
  <your-github-username>/
    1-number-guessing/
    2-task-tracker/
    ...
```

You write all your code inside **your own folder** under `submissions/`. You never edit anyone else's folder or the files in `projects/`.

## The rules

1. Work only inside `submissions/<your-username>/`.
2. One pull request per project.
3. No external npm packages unless the project brief explicitly allows it.
4. Don't commit `node_modules/` or your data files (`tasks.json`, `expenses.json`) — they're already in `.gitignore`.
5. Be respectful of classmates' work — this is a shared space and everyone can see it.

## Getting started

Read **[SUBMISSION-GUIDE.md](SUBMISSION-GUIDE.md)** for the exact step-by-step on branching, adding your code, and opening a pull request.

The full learning guide for each project (with documentation links and worked examples) is handed out separately as the **Student Guide**.
