# Project 1 — Number Guessing Game

A CLI game: the computer picks a secret number from 1–100; you guess until you get it or run out of chances.

## Requirements
- Welcome message + rules at the start.
- Difficulty selection: Easy (10 chances), Medium (5), Hard (3).
- After each guess, say whether the secret is higher or lower.
- Win message with the number of attempts; lose message when chances run out.

## Run it
```bash
node game.js
```

## Rules
- No external npm packages. Built-in modules only (`readline`, etc.).
- Handle non-numeric and out-of-range input gracefully.
