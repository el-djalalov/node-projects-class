# Project 5 — Unit Converter (Web App)

A web page that converts length, weight, and temperature. The user enters a value, picks "from" and "to" units, clicks Convert, and sees the result.

## Requirements
- Sections/tabs for Length, Weight, Temperature.
- An input, two unit dropdowns (from/to), and a Convert button.
- A result view showing the converted value, with a Reset button.

## Run it
```bash
node server.js     # then open http://localhost:3000/length
```

## Rules
- Build the server with Node's built-in `http` module (Express allowed if your instructor okays it).
- Length and weight convert by a base-unit factor; temperature needs formulas.
- Match the provided wireframe layout.
